# Project Demonstration

## Steps to Demonstrate
1. Open the project in VS Code and ensure all project folders are imported.
2. Run `app.py` and click on the server URL shown in the terminal to open the application in your browser.

## Home Page
The home page is displayed upon launching the application — introduces "Opti Crop Agricultural Optimisation."

## About Page
Clicking "About" in the navigation redirects to the About page, which explains the objectives of intelligent crop prediction using environmental and soil parameters, and how machine learning techniques improve agricultural productivity, optimize farming decisions, and support sustainable agricultural practices.

## FindYourCrop Page
Clicking "Find Your Crop" redirects to the prediction form ("Find Your Best Crop Here — Please enter the conditions"). Users enter:
- nitrogen
- phosphorous
- potassium
- temperature
- humidity
- ph
- rainfall

Clicking "Predict" returns the recommended crop, e.g.:
> "Best crop for given conditions is coffee"
