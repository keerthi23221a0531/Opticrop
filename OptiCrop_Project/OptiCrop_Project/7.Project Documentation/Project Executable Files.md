## Run Order
1. `pip install -r requirements.txt`
2. Place `Crop_recommendation.csv` in the working directory
3. `python 1_data_preprocessing.py`
4. `python 2_model_building.py`
5. `python app.py`
6. Open the local server URL shown in the terminal
