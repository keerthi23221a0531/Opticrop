## Problem Statement

Farmers and agricultural planners currently lack an accessible, data-driven tool that can recommend the most suitable crop for a given set of soil and environmental conditions. This leads to suboptimal crop choices, reduced yields, inefficient use of resources, and increased financial risk for farmers, especially those with limited access to agronomic expertise.

**OptiCrop** addresses this by using machine learning (K-Means clustering + Logistic Regression) trained on historical agricultural data to recommend the most suitable crop based on Nitrogen, Phosphorous, Potassium, temperature, humidity, pH, and rainfall — delivered through a simple, accessible web application.
